# Security Policy

## Supported versions

This is a portfolio/demo project; only the latest code on the `main` branch is supported.

| Version | Supported |
| ------- | --------- |
| `main`  | ✅         |

## Reporting a vulnerability

Please report security vulnerabilities **privately** by email to
**contact@ferzendervarli.com**.

- Include a description, steps to reproduce, and the potential impact.
- Do **not** open a public issue or disclose the vulnerability publicly before it has
  been acknowledged and addressed.
- You can expect an acknowledgement within a few business days, followed by a fix or
  mitigation plan.

Thank you for helping keep the project and its users safe.
